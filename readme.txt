Basic project structure

Assets/:
- Scripts/:
  - Utils/: Основные скрипты, которые используются практически повсеместно.
  - UI/: Скрипты для пользовательского интерфейса.
  - Gameplay/: Скрипты, связанные с игровым процессом.
  - Models/: Содержит классы данных (модели) и менеджеры данных.
  - Services/: Сервисы (например, управление загрузкой, звуком).
  - Managers/: Менеджеры для управления различными системами игры.
Scenes/:
- MainMenu: Сцены главного меню.
- Game: Сцены игрового процесса.
Prefabs/: Шаблоны игровых объектов.
Resources/: Общие ресурсы, которые могут загружаться на любом этапе, такие как текстуры, звуки и т.д.
Materials/: Материалы для 3D моделей.
Models/: 3D модели и FBX файлы.
Textures/: Текстуры и спрайты.
Animation/: Анимации и аниматоры.
